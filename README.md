 ECHO Music — Spotify-Inspired Web UI

A sleek, modern music player interface built with clean HTML5, CSS3, and Vanilla JavaScript. **ECHO Music** features a dark-mode aesthetic with a signature electric cyan accent and responsive web component layouts.


 Features

Dark Theme UI:** Designed with CSS custom properties  for clean layout management.
Dynamic Greetings:** Automatic time-based greeting updates based on local system time.
Interactive Control Bar:** Functional play/pause button state toggles and styled volume/progress indicators.
Hover-Animated Cards:** CSS Grid layout with smooth hover transformations and quick-play button reveals.
Lightweight:** Built with zero frameworks or heavy build tools—just standard HTML, CSS, and Vanilla JS + FontAwesome icons via CDN.
